# 微服务公用组件

微服务中一些公共配置放在该模块中。

## 包及作用

### annotation

存放自定义注解

### aop

配置全局的aop

### bo

业务对象（大对象），暂时不使用

### config

公用配置

### constant

微服务中的常量类

### dto

DTO对象

### entity（弃用）

不用

### enumerate

公共枚举类

### exception

存放微服务中的异常相关的信息

### openfeign

存放openFeign的接口

### po

实体类，对应数据表

### pojo

普通的java对象

### scheduler

微服务中定时器

### utils

工具类

### validated（弃用）

原使用接口来限制参数，现新建对象来限制

### vo

控制器接参对象，也是微服务之间调用对象
